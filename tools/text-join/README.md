# 字符拼接工具

按分隔符拆分每行原始文本，用 `$1`、`${1}`… 占位符模版批量生成目标字符串。

## 用法

1. 在「原始文本」中每行输入一条记录（如 `10.1.1.1,80`）
2. 设置字段分隔符（默认逗号），可点击快捷按钮选择
3. 在「模版文本」中编写模版，如 `create ip ${1} port ${2};`
4. 勾选「实时转换」时输入变化会自动更新结果；也可点击「转换」手动执行

## 模版管理

- **历史记录**：转换成功时自动保存最近 20 条模版（同文本去重后置顶）
- **固定模版**：点击「固定当前模版」将当前模版持久保存到浏览器 localStorage，不参与 20 条淘汰；可填写备注便于识别
- 点击历史或固定条目可快速载入；固定模版在 Service Worker 缓存更新后仍会保留

## 占位符规则

- 同时支持 `$1` 与 `${1}` 两种写法，均表示第 1 个字段（`$2` / `${2}` 同理）
- 建议当模板中需要出现普通 `$` 文本时优先使用 `${n}` 变量语法，避免歧义
- 若需要强制输出字面量 `$`，可使用转义：`\$`（例如 `\$1` 会输出 `$1`）
- 索引越界时保留原占位符（如 `$99`、`${99}` 不替换）
- 分隔符为空时整行作为 `$1`；空格、Tab 等纯空白分隔符可正常使用

## 示例

| 原始文本 | 分隔符 | 模版 | 输出 |
|---------|--------|------|------|
| `10.1.1.1,80` | `,` | `create ip ${1} port ${2};` | `create ip 10.1.1.1 port 80;` |
| `web01 prod` | 空格 | `host=$1 env=$2` | `host=web01 env=prod` |
