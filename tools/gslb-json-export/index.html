📘 使用说明
【GSLB JSON 导出工具 使用说明】 工具用途： 1. 用于解析 GSLB 配置 JSON 文件（包含域名、地址池、成员信息），并导出为带中文表头的 CSV 文件。 2. 适用于运维巡检、故障排查、配置核对、变更前后比对等场景。 功能特性： 1 支持域名、地址池、成员字段自由选择与排序 2 支持字段方案记忆（自动记忆最近配置，保存在浏览器本地） 3 支持高级字段：HMS、Pass、池启用、DC启用等 4 支持全量预览与实时过滤，支持 Ctrl+C 复制选中行 5 点击表格行选择域名，再查看该域名的「域名 → 地址池 → 服务成员」引用关系图（节点与连线展示关键参数） 6 导出 CSV 使用 UTF-8 BOM，可直接用 Excel 打开 操作步骤： 1. 登录设备管理界面，依次点击解析管理 → 数据管理，点击「导出」，数据类型选择「动态DNS」，导出类型选择「全部数据导出」，点击确认后完成下载 2. 将步骤 1 下载的文件导入至工具，点击「导入 JSON」选择源文件 3. 在字段配置区选择需要导出的字段，可排序 4. 点击「预览」查看表格；点击某行选择域名，再点「查看关系图」或双击行查看引用拓扑 5. 点击「导出 CSV」生成文件 导出说明： 1 可导出域名关联的对象数据 2 按选中字段顺序导出列 建议使用场景： 1 运维巡检：检查池、成员状态与 HMS 配置 2 故障排查：快速定位异常成员/池/域名 3 配置核对：与变更单或预案对比
